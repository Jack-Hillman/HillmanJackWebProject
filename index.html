Why Everyone Should Know how to Build a Computer

Computers offer users endless information, pass times, and more! As time moves on, we have seen computer become more and more important in our daily lives. They are used in the work place, at homes, and even in cars! This is all the more reason to understand the basics of how they work. Understanding everything about computers in nearly impossible, but hopefully I'll be able to explain the basic functions of the seven main parts and how they work together to form the basis of a computer.

What are the Seven Main Components?

  • Graphics Processing Unit(GPU)
  • Central Processing Unit(CPU)
  • Random Access Memory(RAM)
  • Storage Drive
  • Power Supply
  • Motherboard
  • Case

These components all work together to run the computer that you are currently using to view this website. For instructions on how to build the computer, click here, and for a description of each part and how it works, click here.
